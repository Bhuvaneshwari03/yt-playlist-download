# YT Playlist Download

A tool to download YouTube playlists.

## Features

- Download full playlists.
- Support for various formats.
- Metadata preservation.

## Getting Started

### Prerequisites

- Python 3.x
- `yt-dlp`

### Installation

```bash
pip install -r requirements.txt
```

### Usage

```bash
python main.py <playlist_url>
```
